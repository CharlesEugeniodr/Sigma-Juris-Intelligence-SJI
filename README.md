## 📋 Sumário Executivo

O **Sigma—Juris Intelligence Framework (SJIF)** é um Sistema Integrado de Inteligência Jurídica projetado como um tratado metodológico completo. Sua arquitetura é análoga à de sistemas operacionais e plataformas ERP, garantindo robustez, escalabilidade e interconexão entre todos os componentes.

### Missão
Capacitar profissionais do direito com ferramentas analíticas e metodológicas avançadas, transformando dados jurídicos brutos em **inteligência acionável**.

### Visão
Ser a principal referência em inteligência jurídica especializada, reconhecida pela profundidade metodológica, rigor técnico e capacidade de adaptação às complexidades do ambiente jurídico.

### Diretiva Mestra
> *Nenhuma linha poderá ser ignorada, nenhuma prova omitida, nenhuma decisão desconsiderada e nenhuma jurisprudência relevante poderá deixar de ser pesquisada.*

## ⚙️ Componentes Principais

### 🧠 Kernel Mestre Jurídico (KMJ)
O cérebro orquestrador do SJIF, responsável por coordenar todos os motores e módulos. Composto por:
- **Motor de Inferência Lógica** — Raciocínio jurídico formal
- **Gerenciador de Contexto** — Estado da análise em andamento
- **Módulo de Resolução de Conflitos** — Antinomias e hierarquia normativa
- **API Interna** — Comunicação padronizada entre componentes
- **Módulo de Aprendizado Adaptativo** — Melhoria contínua baseada em feedback

### 🔧 23+ Motores Especializados

| Categoria | Motores |
|-----------|---------|
| **Engenharia** | Processual · Prova · Fundamentação · Pedidos · Reversa · Recursal · Execução |
| **Pesquisa** | Normativo · Jurisprudencial · Doutrinário · Benchmark · Comparado |
| **Estratégia** | Estratégico · Riscos · Compliance · Auditoria · Coerência · Decisório |
| **Técnicos** | Matemático · Estatístico · Simulação · Negociação · Constitucional |
| **Por Área** | Empresarial · Tributário · Trabalhista · Minerário · Ambiental · Agrário |

### 📖 6 Bibliotecas

| Biblioteca | Conteúdo |
|-----------|----------|
| **Jurídica** | 16 áreas do Direito (Civil, Empresarial, Tributário, etc.) |
| **Briefings** | Briefing Mestre + 20 especializados (50–300 perguntas cada) |
| **Templates** | 14 categorias (Petições, Contratos, Pareceres, etc.) |
| **Checklists** | 13+ tipos (Due Diligence, Compliance, Processual, etc.) |
| **Indicadores** | KPIs (8 categorias) + KRIs (4 categorias) |
| **Estratégias** | 5 tipos de estratégia com plano de ação de 10 componentes |

### 📊 10 Modelos Matemáticos
Ponderação · Probabilístico · Multicritério · Bayesiano · Sensibilidade · Priorização · Peso das Provas · Robustez da Fundamentação · Risco Jurídico · Simulação de Cenários

### 🤖 Inteligência Artificial
- **PLN** — Extração de Informação, Análise de Sentimento, Geração de Linguagem
- **Machine Learning** — Supervisionado, Não Supervisionado, Reforço
- **Deep Learning** — Redes Neurais para reconhecimento de padrões
- **Sistemas Especialistas** — Rule Engines para raciocínio baseado em regras

## 🌐 Aplicação Web

O SJIF inclui uma **aplicação web completa** com frontend SPA e backend API:

| Componente | Descrição |
|---|---|
| **`app/`** | Frontend SPA com vanilla JavaScript, Dashboard com Chart.js, Classificador de Documentos (197 tipos), Analisador (9 elementos), MIJ Dashboard, Magistrados e Simulador |
| **`server/`** | Backend API com FastAPI, SQLAlchemy (async), JWT + bcrypt auth, Motor de Inteligência Judicial (MIJ) com métricas SET/IPT/TR/IAJ, scrapers para DataJud/TJMG/STJ |
