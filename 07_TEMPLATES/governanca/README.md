# 🏛️ Governança Corporativa

> **Diretório**: `07_TEMPLATES/governanca/`
> **Categoria**: Templates de Governança

## Propósito

Fornecer modelos padronizados para a documentação de práticas de governança corporativa, incluindo atas de reunião, estatutos sociais, regimentos internos, procurações e relatórios de governança. Os templates asseguram a conformidade com as melhores práticas de governança, a transparência dos processos decisórios e a documentação adequada dos atos societários.

## Tipos de Documento

| Tipo | Descrição |
|---|---|
| **Ata de Reunião do Conselho** | Registro formal das deliberações do conselho |
| **Ata de Assembleia** | Registro de assembleias de sócios/acionistas |
| **Estatuto Social** | Instrumento constitutivo da sociedade |
| **Contrato Social** | Constituição de sociedades limitadas |
| **Regimento Interno** | Normas de funcionamento de órgãos societários |
| **Procuração** | Instrumentos de mandato empresarial |
| **Relatório de Governança** | Relatório periódico de práticas de governança |
| **Termo de Posse** | Posse de conselheiros e diretores |
| **Política de Transações com Partes Relacionadas** | Regulamentação de operações com partes relacionadas |
| **Política de Divulgação** | Diretrizes para divulgação de informações |

## Estrutura Padrão

```
1. Identificação da Sociedade
2. Órgão/Reunião (Conselho, Diretoria, Assembleia)
3. Data, Hora e Local
4. Participantes / Quórum
5. Ordem do Dia
6. Deliberações
   6.1. Matéria deliberada
   6.2. Discussão
   6.3. Votação e resultado
7. Encerramento
8. Assinaturas
```

## Campos Dinâmicos

```
{{sociedade_nome}}         → Denominação social
{{cnpj_sociedade}}         → CNPJ da sociedade
{{orgao_deliberativo}}     → Órgão (Conselho, Diretoria, Assembleia)
{{data_reuniao}}           → Data da reunião/assembleia
{{local_reuniao}}          → Local de realização
{{presidente_mesa}}        → Presidente da mesa
{{secretario_mesa}}        → Secretário da mesa
{{quorum_presente}}        → Quórum de presença
{{ordem_dia}}              → Ordem do dia
{{deliberacoes}}           → Deliberações tomadas
```

## Integração com Motores

- **Motor Empresarial** — Contextualização no ambiente corporativo
- **Motor de Compliance** — Conformidade com normas de governança
- **Motor Normativo** — Atualização conforme Lei das S.A., CC e CVM
- **Motor de Riscos** — Identificação de riscos societários
- **MCJ (Cap. 23)** — Verificação de conformidade dos atos
